Anleitung & Veröffentlichung auf GitHub Pages

So funktioniert's: Lehrperson wählt Sprache und Niveau, gibt ein Thema und einen Lerntext ein und startet. Der Schüler tippt auf das Mikrofon, spricht, und die App antwortet gesprochen – sie stellt Fragen zum Text, lobt, fragt nach, wenn der Schüler nicht weiterweiß, und wiederholt eine korrigierte Fassung nur dann, wenn sprachlich etwas verbessert werden sollte.

Browser: Spracherkennung läuft am zuverlässigsten in Google Chrome (Desktop und Android). Mikrofonzugriff muss erlaubt werden.

Auf GitHub Pages: Lade diese Datei als index.html in ein Repository, aktiviere unter Settings → Pages die Veröffentlichung. Dort erscheint dann das Feld für deinen API-Key, weil der Browser direkt mit der Anthropic-API spricht. Den Key bekommst du unter console.anthropic.com.
